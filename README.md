# teal_wallpaper dotfiles

Hyprland rice · Teal & Pink Anime · Sharp Minimalist

## Stack
- Hyprland + Hyprpaper
- Waybar (floating pill)
- Rofi
- Kitty
- Fastfetch

## Palette
| Role        | Hex       |
|-------------|-----------|
| Background  | `#6BBFB5` |
| Surface     | `#EFF8F7` |
| Pink accent | `#F2A7B8` |
| Text        | `#2C3E3D` |

## Install
Clone the repo and symlink the configs:
```bash
git clone https://github.com/haram0653-hub/teal_wallpaper.git
cd teal_wallpaper/dotfiles
stow --target=$HOME .
```
